## Вычисление интеграла методом Монте-Карло

# Запуск

В заголовочном файле можно выбрать кол-во потоков и точек, что характеризует точность интеграла и время его вычисления.
Чтобы запустить программу:

``` bash
  Make
```
``` bash
  sudo ./integral.out
```
